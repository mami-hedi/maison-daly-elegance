# Maison Daly Elegance

Site de réservation en ligne pour une maison d’hôtes.  
Développé avec **React/TypeScript** côté client et **Node.js/Express** côté serveur, avec une base de données MySQL.

## Fonctionnalités

- Recherche de disponibilité par dates.
- Affichage des chambres disponibles.
- Réservation d’une chambre avec formulaire complet.
- Vérification automatique des dates réservées.
- Gestion des réservations côté serveur.
- Affichage visuel des jours réservés dans le calendrier.
- Redirection vers la page de réservation de la chambre avec les dates sélectionnées.

---

## Technologies

- **Frontend** : React, TypeScript, React Router, React Datepicker, TailwindCSS
- **Backend** : Node.js, Express, TypeScript
- **Base de données** : MySQL
- **Outils** : Vite, npm/yarn, Git
